# Saudi-Arabia-sales
A simple sales dashboard built to analyze sales performance. It shows total sales, average sales, monthly trends, product category performance, and sales distribution by city. The dashboard helps identify key insights and supports data-driven business decisions.  Tools: Power BI, Data Analysis, Data Visualization 📊
🇸🇦 Saudi Arabia Sales Analysis - Power BI
📋 وصف المشروع (Project Overview):
تحليل شامل لبيانات المبيعات في المملكة العربية السعودية باستخدام Power BI. يهدف المشروع إلى تتبع الأداء المالي، تحديد المناطق الأكثر مبيعاً، وفهم سلوك العملاء لاتخاذ قرارات مبنية على البيانات.
🛠️ الأدوات المستخدمة (Tools Used):
Power BI Desktop: لتصميم الداشبورد ومعالجة البيانات.
Excel / CSV: كمصدر للبيانات الأساسية.
DAX Measures: لحساب المؤشرات الرئيسية (KPIs) مثل إجمالي الأرباح ونسبة النمو.
💡 أهم النتائج (Key Insights):
تحديد أكثر المناطق تحقيقاً للأرباح.
تحليل أداء المنتجات المختلفة ومقارنتها.
مراقبة المبيعات الشهرية وتحديد فترات الذروة.
